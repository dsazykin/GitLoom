# GitLoomOS payload — package set (P2-21 invariant 2: tarball hash-stable given pinned inputs).

# Versions are NOT pinned per-line. Determinism comes from pinning apt at a frozen
# snapshot.debian.org timestamp (see DEBIAN_SNAPSHOT in the Dockerfile): the snapshot freezes the
# whole bookworm archive, so installing a package *name* yields the one deterministic version that
# existed at that instant. That is both bit-reproducible AND always fetchable — exact per-version
# pins used to rot the moment Debian retired an old version from the mirrors.
#
# To move the version floor, bump DEBIAN_SNAPSHOT in the Dockerfile (see docs/gitloomos-updates.md).
